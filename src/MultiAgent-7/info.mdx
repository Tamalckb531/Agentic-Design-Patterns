# Multi Agent system:
A process of involving multiple agent to done a complex task cause, single agent process will be lengthy and can cause more impurity in result. 

## Various form of Multi agent setup:
1. Sequential Handsoff ->  one agent complete a task and send the output to another agent through the pipeline to complete further task.
2. Parallel Processing -> Multiple agent works simultaneously to complete a single task 
3. Debate and Consensus -> Multiple agent discuss and take a decision
4. Hierarchical Structures -> Manager worker agent system
5. Expert Team -> Agent with specialized knowledge in domain collaborates
6. Critic-Reviewer -> Reflexive type agent
