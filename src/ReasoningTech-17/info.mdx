# Reasoning Techniques
Reasoning Techniques provide the "Internal Monologue" for an agent. Instead of jumping from a prompt to a final answer, the agent follows structured logical paths.

# Main Constraints 
1. Chain of Thought (CoT)
2. Tree of Thoughts (ToT)
3. ReAct

# How to use
1. Implement "Self-Correction"
2. Enforce "Intermediate Steps"
3. Multi-Path Planning
